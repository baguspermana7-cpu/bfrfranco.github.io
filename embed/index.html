Embedding Guidelines

  • All widgets are free to embed on any website -- commercial or personal, no attribution beyond the built-in "Powered by ResistanceZero" badge is required
  • Each widget is fully self-contained with inline CSS and JavaScript -- no external stylesheets or libraries needed
  • Widgets use a dark theme by default (#0f172a background, #06b6d4 cyan accents) that looks great on both light and dark sites
  • Minimum width: 300px -- all widgets are responsive and adapt to narrow containers
  • Calculations run entirely client-side -- no data is sent to any server. Only anonymous Google Analytics pageview events are tracked
  • For the full-featured versions with advanced inputs, charts, and PDF export, visit the main calculator pages linked in each widget footer
  • Need a custom widget or whitelabel version? Contact us via the main site
