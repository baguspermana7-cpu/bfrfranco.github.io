    Engineering Mode Guide
    This page emulates a compact BMS/HMI for air-cooled chiller plant hydraulics. Use this guide to run controlled drills and understand how each control affects flow, dT, and pressure envelope.
    SIM Toggle

    Use: Freeze or run live simulation updates.

    Impact: When OFF, values hold and flow animations pause. Useful for static review and screenshot audits.

    MAINT Toggle

    Use: Enable maintenance PID settling around setpoints.

    Impact: Pump speed and hydraulic values converge to CHWS/DP/Flow targets, reducing random drift.

    CHWS SP and DP SP

    Use: Set thermal and pressure operating targets.

    Impact: Lower CHWS SP increases cooling effort; higher DP SP raises pump workload and flow margin.

    Flow SP and Motion

    Use: Flow SP tunes loop demand in MAINT mode, Motion adjusts visualization intensity.

    Impact: Flow SP changes branch thickness and pump speed response; Motion changes dash/impeller speed only.

    Print Mode + Export PNG

    Use: Freeze animation effects into clean engineering snapshot for reporting.

    Impact: Glow/particles are reduced, line clarity improves, and export supports both raw PNG and A4 framed report.

    Fault Injection

    Use: Force branch disconnect for drills.

    Impact: Forced loop enters broken/no-flow envelope, alarm strips become active, and guidance updates.

    Loop Diagnostics

    Use: Click any loop or sidebar card for detailed signals and operator actions.

    Impact: Supports branch-by-branch triage without leaving main mimic context.
