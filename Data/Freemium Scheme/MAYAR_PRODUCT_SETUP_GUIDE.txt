**ResistanceZero PRO — Data Center Intelligence Platform**

Platform analisis infrastruktur data center terlengkap di Indonesia.
Akses kalkulasi profesional, artikel premium, dan insight teknis mendalam
untuk pengambilan keputusan investasi yang lebih cerdas.

**Apa yang Anda dapatkan dengan PRO:**

**Interactive Calculators (Full Access)**
- CAPEX Calculator — estimasi biaya pembangunan data center hingga 100 MW,
  30+ negara, 19 city market pricing, advanced soft cost modeling
- OPEX Calculator — estimasi biaya operasional tahunan dengan breakdown
  12 role staffing, 6 sistem maintenance, utilities, compliance
- Semua parameter unlocked: seismic zone, cooling type, redundancy level,
  generator config, fire suppression, dan puluhan opsi lainnya
- Advanced Mode: projection year, city $/W pricing, front-of-meter costs,
  delivery method, contractor availability, fee sliders

**Premium Analytics & Metrics**
- Cost per kW ($/kW) — metrik standar industri yang dipakai investor global
- Annual Energy Cost — estimasi biaya listrik tahunan detail
- Full Cost Breakdown — semua komponen dengan nilai dollar, persentase,
  dan progress bar (bukan hanya 3 kategori seperti versi Free)
- Staffing breakdown per role (12 posisi), maintenance per sistem (6 sistem),
  utilities per kategori (5 item), insurance & compliance per item

**Comparison & Export**
- Scenario A vs B comparison — bandingkan dua konfigurasi side-by-side
- PDF Export — laporan profesional berformat PDF untuk presentasi klien
- Save & Load Projects — simpan hingga 5 konfigurasi, akses kapan saja
- Tanpa watermark "Free estimate"

**Premium Articles & Technical Analysis**
- Akses artikel premium tentang desain, operasi, dan optimasi data center
- Deep-dive technical analysis: cooling strategy, power distribution,
  sustainability, AI infrastructure requirements
- Geopolitical analysis dampak regulasi terhadap industri data center
- Case study proyek data center di Asia Tenggara dan global
- Konten baru ditambahkan secara berkala

**Indonesia-Specific Intelligence**
- IDR salary baseline untuk 12 role data center
- K3 Listrik compliance cost breakdown
- BPJS calculation dalam staffing model
- SLO/AMDAL compliance listing dengan estimasi biaya

**Mengapa ResistanceZero PRO?**
Laporan data center dari konsultan internasional (Cushman & Wakefield,
Turner & Townsend, JLL) berharga $2,000 - $5,000 per laporan — berupa
PDF statis tanpa kustomisasi. ResistanceZero PRO memberikan analisis
interaktif, real-time, dan customizable dengan harga 95-99% lebih murah.

E5. Integration — Menghubungkan Mayar ke Website:

  MEKANISME INTEGRASI (saat ini):
  Karena website di-host sebagai static site (GitHub Pages), integrasi
  dilakukan via redirect flow — bukan embedded checkout:

  1. CHECKOUT FLOW:
     User klik "Upgrade to Pro" di website
     → Redirect ke https://bagus-dwi-permana.myr.id/m/resistancezero-pro
     → User isi form + pilih durasi + bayar di Mayar
     → Mayar redirect ke https://resistancezero.com/capex-calculator.html
     → Backend (Cloud Function) menerima webhook dari Mayar
     → Cloud Function verifikasi via Mayar API → update Supabase
     → User login di website → entitlements loaded dari Supabase

  2. WEBHOOK SETUP (di Mayar Dashboard > Settings > Webhook):
     URL: https://[YOUR-CLOUD-RUN-URL]/api/webhook/mayar
     (Cloud Function URL akan dibuat di Phase 1)

     Events yang akan diterima:
     - payment.received → activate subscription
     - membership.memberUnsubscribed → handle cancellation
     - membership.memberExpired → downgrade to free
     - payment.reminder → log for dunning system

  3. VERIFIKASI PEMBAYARAN (karena Mayar tidak punya webhook signature):
     Cloud Function harus call GET /hl/v1/invoice/{id}
     dengan Authorization: Bearer {MAYAR_API_KEY}
     untuk memverifikasi status sebelum activate subscription.

  4. FUTURE (Phase 2) — EMBEDDED CHECKOUT:
     Jika ingin checkout tanpa redirect, bisa pakai Mayar API:
     POST https://api.mayar.id/hl/v1/invoice/create
     → Buat invoice programmatically
     → Tampilkan payment link di modal/iframe
