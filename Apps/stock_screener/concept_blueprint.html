Free Float Logic

  • Formula: Free Float = 100% - Strategic Holdings
  • Strategic types: CP, IB, FD, OT
  • Free-float types: MF, IS, PF, SC
  • ID is conditional (insider verification required)
  • ID >= 20% = strategic
  • ID >= 5% in 2+ companies = strategic heuristic
  • Sub-1% holders assumed free-float (KSEI threshold)
